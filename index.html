The Crisis

Why Current Solutions Fail

Traditional water monitoring is reactive, expensive, and slow. By the time lab results arrive, communities may have been consuming toxic water for weeks.

Lab Bottleneck

ICP-MS and AAS require centralized labs, trained technicians, and days of processing. Each test costs hundreds of dollars, making frequent monitoring impossible for developing regions.

Invisible Threats

Heavy metals like Lead, Cadmium, and Mercury are colorless and odorless. Communities consume contaminated water for years before symptoms appear, causing irreversible neurological damage.

Reactive Only

Existing IoT systems only report current values. They cannot predict future contamination events, leaving zero time for preventive action before a toxic plume reaches water treatment intakes.

System Architecture

Edge-to-Cloud AI Pipeline

A vertically integrated stack from physical sensing to predictive intelligence, designed for real-world deployment in resource-constrained environments.

Edge Layer — NodeMCU ESP8266

TCS3200 color sensor → RGB frequency → Concentration mapping

Cloud Layer — ThingSpeak IoT

Real-time visualization, 8-channel monitoring, <5s latency

ML Layer — Random Forest Classifier

Temporal lag features (Lag1, Lag2) → Predictive risk scoring

Alert Layer — Automated EWS

Threshold ≥ 0.70 triggers email alert 4 days before breach

Hardware Innovation

The prototype uses a sealed dark chamber design to eliminate ambient light interference, ensuring Beer-Lambert Law fidelity. The TCS3200 sensor converts reflected light from colorimetric test strips into precise frequency signals for 8 heavy metals: Cu, Fe, Mn, Zn, Hg, Cr, Pb, Cd.

Microcontroller NodeMCU ESP8266
Sensor Array TCS3200 RGB-to-Frequency
Detection Method Colorimetric Paper Strips
Cloud Platform ThingSpeak (MathWorks)
ML Framework Scikit-Learn Random Forest
Power Supply 6V DC / Portable Battery
Key Innovations

What Makes This Different

Four breakthrough innovations that transform water monitoring from reactive logging into proactive environmental intelligence.

Edge Intelligence

Real-time concentration conversion happens on-device using hardcoded calibration matrices, reducing cloud payload by 70% and enabling offline operation.

Correlation Mapping

Pearson analysis revealed Pb-Cd correlation (r=0.79) and Cu-Mn-Hg clusters, mathematically justifying multi-variable ML over single-sensor thresholds.

Temporal Look-Ahead

Lag1/Lag2 feature engineering with shifted target vectors (Yt+1) transforms classification into prediction, enabling 4-day advance warnings.

Automated EWS

Threshold-triggered email alerts via ThingSpeak API ensure authorities receive actionable intelligence without manual dashboard monitoring.

System Timeline

From Sensor to Alert

A continuous operational loop that transforms physical water samples into automated predictive intelligence within seconds.

T+0s

Sample Collection

Water sample applied to colorimetric test strip. Chemical reaction produces color change proportional to heavy metal concentration.

T+2s

Optical Scan

TCS3200 sensor in sealed dark chamber reads RGB reflectance frequencies. ESP8266 converts frequencies to concentration (mg/L) via calibration matrix.

T+5s

Cloud Upload

Data packetized and transmitted via HTTP to ThingSpeak. 8 fields populated with real-time concentration values for all metals.

T+10s

ML Inference

Google Colab pipeline pulls data. Lag features (t-1, t-2) generated for Mn, Hg, Pb, Cd. Random Forest computes future risk probability.

T+15s

Alert Decision

If P(future contamination) >= 0.70, automated email alert dispatched to authorities. 4-day operational lead time activated.
